Política de Privacidad

Última actualización: 16 de abril de 2026

BotPay Center es una plataforma CRM para la gestión de conversaciones de WhatsApp integrada con el sistema de pagos PayCenter. Esta política describe cómo recopilamos, usamos y protegemos la información de los usuarios.

1. Información que recopilamos

  • Número de teléfono y nombre del contacto que inicia conversación vía WhatsApp.
  • Contenido de los mensajes enviados a través del chatbot.
  • Datos de transacciones de pago (monto, estado, banco) cuando el usuario solicita un cobro.

2. Cómo usamos la información

  • Para gestionar y responder conversaciones a través del CRM.
  • Para procesar solicitudes de pago mediante códigos QR de PayCenter.
  • Para generar reportes internos de atención al cliente.

3. Compartición de datos

No vendemos ni cedemos datos personales a terceros. La información de pago es procesada exclusivamente por PayCenter, nuestra propia plataforma de pagos QR para Bolivia.

4. Retención de datos

Los datos de conversación se conservan mientras la cuenta esté activa. Podés solicitar la eliminación de tus datos escribiéndonos al correo indicado a continuación.

5. Seguridad

Utilizamos autenticación JWT, HTTPS y almacenamiento seguro de contraseñas (bcrypt) para proteger la información.

6. Uso de la API de WhatsApp (Meta)

Este servicio utiliza la API de WhatsApp Business Cloud de Meta para enviar y recibir mensajes. El uso de los datos cumple con los Términos de Servicio de WhatsApp Business.

7. Contacto

Para consultas sobre privacidad escribinos a: admin@botpay.local
